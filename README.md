# chdb-node
[chDB](https://github.com/chdb-io/chdb) nodejs bindings for fun and hacking.

### Status

- Experimental bindings
- Requires [`libchdb`](https://github.com/chdb-io/chdb) on the system

### Build

```bash
npm run libchdb
npm install
npm build
npm run test
```
